# VibeCoding 5개월 커리큘럼 웹페이지

1인 기업가를 위한 **바이브코딩 5개월 커리큘럼**을 소개하는 단일 웹페이지입니다.  
수강생이 브라우저에서 바로 커리큘럼 구조와 월별/주차별 목표를 볼 수 있도록 구성되어 있습니다.

## 구성

- `index.html`  
  - 전체 페이지 구조와 콘텐츠(소개, 월별 커리큘럼, 운영 방식, FAQ)를 담고 있습니다.
- `style.css`  
  - 다크 테마 기반의 스타일 정의 파일입니다. 모바일에서도 보기 좋게 반응형으로 처리되어 있습니다.

## 실행 방법

1. 이 폴더를 연 뒤 `index.html` 파일을 더블 클릭해서 브라우저(Chrome 등)로 엽니다.
2. 필요하다면 호스팅 서비스(예: Netlify, Vercel, GitHub Pages 등)에 이 두 파일만 업로드하면,
   어디서든 접속 가능한 커리큘럼 페이지가 됩니다.

## 커스터마이징

- 텍스트 수정: `index.html` 안의 한국어 텍스트를 그대로 수정하면 됩니다.
- 색상/폰트 변경: `style.css` 상단 `:root` 에 정의된 색상 변수를 바꾸면 전체 톤을 쉽게 변경할 수 있습니다.
